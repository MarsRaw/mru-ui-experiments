# Getting started:

- Clone this repo

```bash
cd mru-ui
git submodule init
git submodule update

git submodule init
git submodule update

git clone https://github.com/MarsRaw/mars-raw-utils-data.git
cd mars-raw-utils-data

git submodule update


```
